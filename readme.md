# GLOBAL LAYOFFS DATA VISUALIZATION DASHBOARD

## Project Overview

This project presents a comprehensive visualization of global layoff trends using Python. The objective is to transform raw layoff data into meaningful visual insights that help identify patterns across companies, industries, countries, and time.

The dashboard demonstrates how data visualization can simplify complex datasets and support better business understanding and decision-making.

---

## Objectives

* Visualize global layoff trends.
* Compare layoffs across companies and countries.
* Analyze industry-wise workforce reductions.
* Examine yearly layoff trends.
* Identify relationships among numerical variables using correlation analysis.

---

### Dataset Features

* Company
* Location
* Total Employees Laid Off
* Date
* Percentage Laid Off
* Industry
* Source
* Company Stage
* Funds Raised
* Country
* Date Added

## Visualizations Created

### 1. Top 10 Companies by Total Layoffs

Displays companies with the highest number of employees laid off.

### 2. Top 10 Countries by Total Layoffs

Compares workforce reductions across countries.

### 3. Top 10 Industries by Total Layoffs

Highlights industries most affected by layoffs.

### 4. Layoffs Trend Over Time

Shows yearly changes in global layoffs.

### 5. Industry Distribution

Displays the distribution of the most represented industries using a pie chart.

### 6. Correlation Heatmap

Visualizes relationships among numerical variables such as total layoffs, percentage laid off, and funds raised.

---

## Key Insights

* A small number of companies accounted for a significant share of layoffs.
* The United States reported the highest number of layoff events.
* Technology and finance-related industries experienced substantial workforce reductions.
* Layoff activity varied considerably across different years.
* The correlation analysis provides insights into relationships between funding and workforce reductions.

---

## Conclusion

This project demonstrates how effective data visualization can uncover meaningful insights from real-world business data. By presenting trends through charts and graphs, the analysis provides a clear understanding of global layoff patterns and supports data-driven decision-making.
